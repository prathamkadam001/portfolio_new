# Weekly Content Calendar

Post once per week and improve one site page per week. Keep it practical and based on real work.

## Week 1

- LinkedIn post: before/after lesson from a business website section.
- Site update: improve `/freelance-web-developer-ahmedabad` with one FAQ.
- Search Console: request indexing for the updated page.

## Week 2

- LinkedIn post: why WhatsApp/contact CTAs matter for small business websites.
- Site update: improve one project case study with clearer challenge, solution, outcome.
- Backlink task: add portfolio link to GitHub profile README.

## Week 3

- LinkedIn post: checklist for Ahmedabad businesses before hiring a website developer.
- Site update: improve `/website-developer-ahmedabad` intro and internal links.
- Backlink task: update LinkedIn Featured section with the service page.

## Week 4

- LinkedIn post: short case study from a project screenshot.
- Site update: add or improve one blog article based on Search Console queries.
- Backlink task: ask one real client for permission to add a footer credit.

## Monthly Review

- Top queries by impressions.
- Pages with impressions but no clicks.
- Pages with clicks but no leads.
- WhatsApp, email, phone, and form-success event counts.
- One page to rewrite next month.
