## Weekly Checks

Every Monday:

- Search Console > Sitemaps: confirm sitemap status is `Success`.
- Search Console > Pages: check newly indexed URLs.
- Search Console > Performance: export queries with impressions.
- GA4 Realtime/Reports: review WhatsApp, email, phone, form, and CTA events.
- Improve one page that has impressions but low clicks.

## What To Improve First

For pages with impressions but low clicks:

- Make title more specific.
- Rewrite meta description around a clear offer.
- Add one proof section or project link.
- Add one FAQ based on real client doubt.
- Add internal links from homepage, services, blog, and footer where relevant.
